* Assumptions:
    * User needs to login/signup from the home page of the website


* API Endpoints:
    * /get-token
    * /api/movies
    * /api/movies/<int:movie_id>
    * /api/movies/<int:movie_id>/ratings
    * api/movies/<int:movie_id>/rate
